Example: Prime Number Checker

def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, int(n**0.5)+1):
        if n % i == 0:
            return False
    return True
This function checks if a number n is prime.
If n is less than or equal to 1, it returns False.
Otherwise, it tests divisibility from 2 up to the square root of n.
If any divisor is found, it returns False, otherwise it returns True.
